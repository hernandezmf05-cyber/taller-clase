# Taller de Lógica Secuencial: Fundamentos de JavaScript

````
1 - Forkear el repositorio
2 - Hacer un commit por cada ejercicio
3 - Al finalizar todos los ejercicios lanzar un Pull Request

````

1.  Necesito un programa que calcule el **total de mi compra** en el supermercado. Pide el precio de tres productos diferentes y muestra el total a pagar.
2.  Para un evento, necesito un banner. Pide el **ancho y el alto** de una pared (en metros) y calcula su **área** para saber cuánto material comprar.
3.  Debo calcular cuánto **perímetro** tiene un potrero rectangular. Pide el largo y el ancho del potrero y muestra el resultado.
4.  Mi app del clima solo muestra la temperatura en grados Fahrenheit. Pide la **temperatura en °F** y conviértela a **grados Celsius**.
5.  Necesito saber la **edad de un usuario**. Pide su año de nacimiento y, asumiendo que el año actual es 2024, calcula cuántos años tiene.
6.  Quiero un script que **salude formalmente** a un cliente. Pide el nombre y el apellido del usuario y muestra en consola el mensaje: "Bienvenido, [Nombre Completo]".
7.  Necesito calcular el **promedio de notas** de un estudiante. Pide las 3 notas (de 0 a 5) y muestra el promedio final.
8.  Debo calcular el **salario semanal** de un empleado. Pide el número de horas trabajadas en la semana y el precio que se paga por hora. Muestra el salario total.
9.  Estoy planeando un viaje y el mapa está en millas. Pide una **distancia en millas** y muéstrala convertida a **kilómetros**. (1 milla = 1.60934 km).
10. Un sistema registra el tiempo de trabajo en minutos. Pide un **total de minutos** y calcula cuántas **horas y minutos** representa (ej. 130 minutos = 2 horas y 10 minutos).
11. Necesito calcular el **precio final** de un producto que tiene descuento. Pide el precio original y el porcentaje de descuento (ej. 20 para 20%) y muestra el precio final a pagar.
12. Se debe calcular el **IVA (19%)** de un producto. Pide el precio base (sin IVA) del producto y calcula cuánto es el IVA y cuál es el precio total.
13. Un cliente quiere saber cuánto **capital final** tendrá en una inversión. Pide el capital inicial, la tasa de interés (anual, ej. 5 para 5%) y el número de años. Calcula el interés simple (Capital * (Tasa/100) * Años) y el total.
14. Necesito calcular el área de un **triángulo**. Pide la base y la altura del triángulo y muestra el resultado.
15. El sistema registró mal los datos. Pide el **nombre** de un usuario y luego su **apellido**. Muestra en consola los datos intercambiados (el valor del nombre en el apellido y viceversa).
16. Necesito un conversor de **divisas simple**. Pide una cantidad en dólares (USD) y, asumiendo una tasa de cambio fija (ej. 1 USD = 3900 COP), muestra el equivalente en pesos colombianos.
17. Para un reporte, necesito saber cuántos **kilogramos** pesa un paquete. El sistema lo da en libras (lb). Pide el peso en libras y muéstralo en kilogramos (1 lb = 0.453592 kg).
18. Pide al usuario un **número de dos cifras** (ej. 47) y muestra la suma de sus dígitos (ej. 4 + 7 = 11).
19. Para un evento se necesita saber cuántos refrescos comprar. Pide el **número de asistentes** y, asumiendo que cada asistente tomará 3 refrescos, muestra el total de refrescos necesarios.
20. Necesito calcular la **velocidad promedio** de un auto. Pide la distancia recorrida (en km) y el tiempo que le tomó (en horas) y muestra la velocidad (km/h).
